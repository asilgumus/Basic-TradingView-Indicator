# BB + RSI Strategy (BB+RSI STR)

BB + RSI STR is a TradingView indicator that combines **Bollinger Bands (BB)** and **RSI** to highlight potential buy opportunities on the chart.

## Features

- RSI-based signal detection  
- Bollinger Bands trend confirmation  
- Visual **Long Entry** labels on the chart  
- Clear and simple labeling for easy interpretation  

## Usage

1. Add the script to TradingView as a new Pine Script indicator.  
2. Adjust the parameters according to your needs:  
   - **RSI Value**: Threshold for RSI signals (default: 20)  
   - **Length**: Bollinger Bands period (default: 20)  
   - **StdDev**: Standard deviation multiplier (default: 2)
